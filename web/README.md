# Formation 360 - Landing Page

Landing informativa para **Formation 360 Soccer & Life Program**, academia de fútbol con coaching integral. Familia Noronha.

## Tecnologías

- **React** + **Vite**
- CSS con variables (paleta del logo: azul oscuro, azul vibrante, naranja)
- Fuentes: Bebas Neue, Outfit (Google Fonts)

## Cómo ejecutar

```bash
cd web
npm install
npm run dev
```

Abre [http://localhost:5173](http://localhost:5173) en el navegador.

## Producción

```bash
npm run build
npm run preview
```

Los archivos estáticos se generan en `dist/`.

## Redes sociales

Edita `src/SocialIcons.jsx` y actualiza las URLs en `SOCIAL_LINKS` con tus perfiles reales de Facebook, Instagram y TikTok.
